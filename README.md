# 추정 분할점수 산출 도우미

대한민국 고등학교에서 **정기시험1·정기시험2·수행평가**의 추정 분할점수를 바탕으로 **최종 추정 분할점수**를 산출하는 웹 도구입니다.

> **면책:** 본 도구는 교사의 추정을 돕는 참고용입니다. 실제 성적 산출 방식·학교 지침과 다를 수 있습니다.

## 기능

### 1. 기본 산출
- 요소별 **반영 비율**과 **만점** 설정 (정기시험 기본 100점, 수행평가 기본 만점=반영비율)
- 각 요소의 분할점수 입력 후 최종 추정 분할점수·환산점 계산
- 수행평가 영역 수 1~4개 설정 가능
- 성취도 체계: **5단계(A~E)** 또는 **5단계+미도달** 선택

### 2. 정기시험 도우미
- 난이도별(쉬움·보통·어려움) × 성취도별(A~E) **예상 정답률 표** (첨부 양식형)
- 세부 모드: 문항구분·문항번호·배점합 자동 표시
- JSON/엑셀 저장, 기본 산출 탭에 자동 적용

### 3. 학기말 최종 분할점수 산출 도우미
- **반×번호 행렬** 또는 **문항별 행** 형식으로 학생 성적 입력
- **정기시험1만 반영한 성취도별 학생 비율** — 정기1 원점수 기준
- **정기1+수행 반영 비율** — 정기1·수행 분할점수의 환산점 합으로 A/B, B/C, … 경계 설정 (만점 = 정기1+수행 반영비율 합)
- **정기시험2 추정분할점수 초안** — 목표 최종 비율 → 정기2 분할점수 역산 (5점 단위)

### 4. 학생 예측
- 학생 점수로 예상 성취도 판정

## 수식

### 통합 환산점 모델

```
환산점 = 분할점수 × (반영비율 ÷ 만점)
최종_XY = 정기1 환산점 + 정기2 환산점 + 수행 환산점  → 소수 첫째 자리에서 반올림하여 정수
```

| 요소 | 기본 만점 | 기본 반영 | A/B 예시 |
|------|-----------|-----------|----------|
| 정기1 | 100 | 30% | 90 × 30/100 = 27.0 |
| 정기2 | 100 | 30% | 88 × 30/100 = 26.4 |
| 수행 | 40 (=반영비율) | 40% | 36 × 40/40 = 36.0 |

→ 최종 A/B = **89** (89.4 → 반올림)

### 정기2 역산

```
정기2_XY = (최종_XY − 정기1 환산점 − 수행 환산점) × 만점2 ÷ 반영비율2
```
