# 추정 분할점수 산출 도우미

대한민국 고등학교에서 **정기시험1·정기시험2·수행평가**의 추정 분할점수를 바탕으로 **최종 추정 분할점수**를 산출하는 웹 도구입니다.

> **면책:** 본 도구는 교사의 추정을 돕는 참고용입니다. 실제 성적 산출 방식·학교 지침과 다를 수 있습니다.

## 기능

### 1. 기본 산출
- 요소별 **반영 비율**과 **만점** 설정 (정기시험 기본 100점, 수행평가 기본 만점=반영비율)
- 각 요소의 분할점수 입력 후 최종 추정 분할점수·기여분 계산
- 성취도 체계: **5단계(A~E)** 또는 **5단계+미도달** 선택

### 2. 정기시험 도우미
- 난이도별(쉬움·보통·어려움) × 성취도별(A~E) **예상 정답률 표** (첨부 양식형)
- 세부 모드: 문항구분·문항번호·배점합 자동 표시
- JSON/엑셀 저장, 기본 산출 탭에 자동 적용

### 3. 정기2 조율
- 정기1·수행이 고정된 상태에서 목표 최종 분할점수에 맞게 정기2 역산

### 4. 학생 예측
- 학생 점수로 예상 성취도 판정

## 수식

### 통합 기여분 모델

```
기여분 = 분할점수 × (반영비율 ÷ 만점)
최종_XY = 정기1 기여 + 정기2 기여 + 수행 기여  → 소수 첫째 자리에서 반올림하여 정수
```

| 요소 | 기본 만점 | 기본 반영 | A/B 예시 |
|------|-----------|-----------|----------|
| 정기1 | 100 | 30% | 90 × 30/100 = 27.0 |
| 정기2 | 100 | 30% | 88 × 30/100 = 26.4 |
| 수행 | 40 (=반영비율) | 40% | 36 × 40/40 = 36.0 |

→ 최종 A/B = **89** (89.4 → 반올림)

### 정기2 역산

```
정기2_XY = (최종_XY − 정기1 기여 − 수행 기여) × 만점2 ÷ 반영비율2
```
