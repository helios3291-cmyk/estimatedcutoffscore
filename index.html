추정 분할점수 산출 도우미 고등학교 시험 배점 도우미

정기시험·수행평가 분할점수와 반영 비율로 학기말 추정 분할점수를 산출합니다.

성취도 체계
초기화
프로필(학생 데이터 제외)

저장된 데이터(학생 데이터 포함)는 브라우저 로컬 저장소(localStorage/sessionStorage)에만 저장되며, 서버로 전송·저장되지 않습니다.

사용 가이드

본 도구는 교사의 추정을 돕는 참고용입니다. 실제 성적 산출 방식·학교 지침과 다를 수 있습니다.

환산점 = 분할점수 × (반영 비율 ÷ 만점)
학기말 점수 = 정기1 환산점 + 정기2 환산점 + 수행 환산점 (정수 반올림)
